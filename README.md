# 💼 Sales Reply Assistant (Free)

A simple tool to help export sales professionals handle difficult customer situations in English.

## 🚀 Features

- Input your situation freely
- Get instant professional replies
- 3 styles:
  - Soft
  - Firm
  - Strategic
- One-click copy

## 🎯 Use Cases

- Price negotiation
- Freight increase
- Customer complaints
- Payment issues

## 🧪 Example

Input:
"Customer says price too high and freight increased"

Output:
✔ Professional responses ready to send
